Caratteristiche del progetto

Integrazione con Elasticsearch

Supporto per la ricerca full-text e gestione dei documenti con un'istanza di Elasticsearch.

API RESTful

Interagisci con il sistema di indicizzazione tramite endpoint ben definiti.

Tecnologie moderne

Sviluppato con Spring Boot, Java e Bootstrap per un'esperienza utente fluida.

Come iniziare

  • Scarica e avvia un'istanza di Elasticsearch.
  • Configura il file application.properties per connetterti all'istanza.
  • Avvia l'applicazione con mvn spring-boot:run.
  • Prova le API utilizzando strumenti come Postman o cURL.
